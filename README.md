# KPSpatial-release

<img src="assets/kpspatial_schematic.png" alt="Project Logo" width="200">

Code &amp; scripts for reproducing spatial-lineage analyses in Jones*, Sun* et al. ([manuscript here](https://www.biorxiv.org/content/10.1101/2024.10.21.619529v2)) 

`./scripts/`: Scripts for processing data.
`./utilities/`: Utilities for handling spatial and phylogenetic data.
`./reproducibility/`: Notebooks to reproduce each figure panel.

To run these notebooks, clone this repository and add the path to your system path to run utilities.
You may find the processed datasets on Zenodo, accession number...
